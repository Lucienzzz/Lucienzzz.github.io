1. 稳定性

​ 稳定性就是排序前后，相同元素所在的相对位置不变。

​ 冒泡 插入 归并 ——–>稳定

​ 选择 快排 堆排 ——–>不稳定

2. 工业界的综合排序（简单说明）

​ 基本数据类型：快排 （基本类型不需要稳定性）

​ 自定义类型（结构体，类）： 归并 （可能要按照多个标准来排序，所以需要）（在某些递归范围小于60直接插排）

​ 数组很短 （一般在60以下，O（N^2）劣势表现不出来，而且常数项极少）： 插排

3. 排序问题的补充

​ 1.归并排序额外空间复杂度可以变成O（1），但是非常难，可以用归并排序内部缓存法

​ 2.快排可以做到稳定性，但是非常难，可以收 “01 table sort”

4. 比较器

​ 在面向对象语言中，涉及到复杂数据类型的时候，比如一个类有多个属性，我们要根据这个类的某一个属性进行排序，这时候就要用比较器，告诉高级语言，我们要用哪个属性的升序还是降序排列。

以优先队列（底层实现：堆），TreeMap（底层红黑树）为例，代码如下：

import java.util.Comparator;
import java.util.PriorityQueue;
import java.util.TreeSet;

public class 简单的比较器{


    public static class studentComparator implements Comparator<Student>{

        @Override
        public int compare(Student o1, Student o2) {
            return o1.id-o2.id;
        }

    }

    public static void main(String[] args) {
        Student student1 = new Student("zhangsan",1,12);
        Student student2 = new Student("lisi",2,12);
        Student student3 = new Student("wangwu",3,10);
        PriorityQueue<Student> heap = new PriorityQueue<Student>(new studentComparator());
        heap.add(student1);
        heap.add(student2);
        heap.add(student3);

        while (!heap.isEmpty()){
            Student poll = heap.poll();
            System.out.println(poll.name + " " + poll.id + " " + poll.age);
        }

        TreeSet<Student> tree = new TreeSet<>(new studentComparator());

    }
}
