1. 稳定性

​ 稳定性就是排序前后，相同元素所在的相对位置不变。

​ 冒泡 插入 归并 ——–>稳定

​ 选择 快排 堆排 ——–>不稳定

2. 工业界的综合排序（简单说明）

​ 基本数据类型：快排 （基本类型不需要稳定性）

​ 自定义类型（结构体，类）： 归并 （可能要按照多个标准来排序，所以需要）（在某些递归范围小于60直接插排）

​ 数组很短 （一般在60以下，O（N^2）劣势表现不出来，而且常数项极少）： 插排

3. 排序问题的补充

​ 1.归并排序额外空间复杂度可以变成O（1），但是非常难，可以用归并排序内部缓存法

​ 2.快排可以做到稳定性，但是非常难，可以收 “01 table sort”

4. 比较器

​ 在面向对象语言中，涉及到复杂数据类型的时候，比如一个类有多个属性，我们要根据这个类的某一个属性进行排序，这时候就要用比较器，告诉高级语言，我们要用哪个属性的升序还是降序排列。

以优先队列（底层实现：堆），TreeMap（底层红黑树）为例，代码如下：

import java.util.Comparator;

5. 桶排序

import java.util.Arrays;

public class 最大间隙MaxGap {
    public static int maxGap(int a[]) {
        if(a.length < 2 || a == null){
            return 0;
        }
        int max = Integer.MIN_VALUE;
        int min = Integer.MAX_VALUE;
        for (int i = 0; i < a.length; i++) {
            max = Math.max(max,a[i]);
            min = Math.min(min,a[i]);
        }
        if(min == max){
            return 0;
        }
        int maxs[] = new int[a.length+1];
        int mins[] = new int[a.length+1];
        boolean hasNum[] = new boolean[a.length+1];
        for (int i = 0; i < a.length; i++) {
            int pos = getGap(a,max,min,a[i]);
            maxs[pos] = hasNum[pos]?Math.max(maxs[pos],a[i]):a[i];
            mins[pos] = hasNum[pos]?Math.min(mins[pos],a[i]):a[i];
            hasNum[pos] = true;
        }
        //求Maxgap
        int g_Max = maxs[0];
        int temp = 0;
        for (int i = 1; i < a.length+1; i++) {
            if(hasNum[i]){
                temp = Math.max(temp,mins[i]-g_Max);
                g_Max = maxs[i];
            }
        }
        return temp;
    }

    private static int getGap(int[] a, int max, int min, int num) {
        return (num - min)*a.length/(max - min);
    }

7. 用数组结构实现队列和栈

太简单了，直接贴代码。tip：用size可以控制大小，可以减少start和end的耦合性

public class 栈和队列 {
    public static class stack{
        private int arr[];
        private int index;
        private stack(int initsize) throws Exception {
            if(initsize < 0)
                throw new Exception("初始化大学非负数");
            arr = new int[initsize];
            index = 0;
        }
        private int pop() throws Exception {
            if(index == 0)
                throw new Exception("已经没有元素了");
            return arr[--index];
        }
        private int peek() throws Exception {
            if(index == 0)
                throw new Exception("没有元素了哇");
            return arr[index-1];
        }
        private void push(int obj) throws Exception {
            if(index == arr.length)
                throw new Exception("已经满了");
            arr[index++] = obj;
        }
        private boolean isEmpty(){
            return index==0;
        }
    }

    public static class quene{
        private int arr[];
        private int start;
        private int end;
        private int size;
        private quene(int initsize) throws Exception {
            if(initsize < 0)
                throw new Exception("初始化大小不能为负数");
            arr = new int[initsize];
            size = start = end = 0;
        }
        private void push(int obj) throws Exception {
            if(size == arr.length){
                throw new Exception("队列已经满了");
            }
            arr[end] = obj;
            end = end==arr.length-1?0:end+1;
            size++;
        }
        private int pop() throws Exception {
            if(size == 0){
                throw new Exception("没有元素了");
            }
            --size;
            int temp = arr[start];
            start = start == arr.length-1?0:start+1;
            return temp;
        }
    }

    public static void main(String[] args) throws Exception {
        stack stack = new stack(3);
        stack.push(10);
        stack.push(3);
        stack.push(-1);
        while (!stack.isEmpty()){
            System.out.println(stack.pop());
        }
        stack.peek();
        quene quene = new quene(3);
        quene.push(3);
        quene.push(6);
        quene.push(10);
        quene.pop();
        quene.push(5);
        System.out.println(quene.pop());
        System.out.println(quene.pop());
        System.out.println(quene.pop());
        Queue<Integer> integers = new ArrayBlockingQueue<Integer>(3);
        integers.add(3);
        integers.add(6);
        integers.add(10);
        integers.poll();
        integers.add(5);
        while (!integers.isEmpty()){
            System.out.println(integers.poll());
        }
    }
}

9. 用队列结构实现栈

用的双向队列，有n个元素，要返回peek，要n-1个元素放入另外一个help队列中，然后返回这第n个元素，交换help和data的索引，也就是始终有一个空队列可以转化最后一个元素，从而达到实现栈的目的。

import java.util.LinkedList;
import java.util.Queue;

public class 双队列实现栈 {
    public static class twoQueneStack<I extends Number> {
        private Queue<Integer> data;
        private Queue<Integer> help;

        public twoQueneStack() {
            this.data = new LinkedList<>();
            this.help = new LinkedList<>();
        }

        public void push(int obj){
            data.add(obj);
        }
        public int peek(){
            while (data.size() > 1){
                help.add(data.poll());
            }
            int temp = data.poll();
            help.add(temp);
            swap();
            return temp;
        }
        public int poll(){
            while (data.size() > 1){
                help.add(data.poll());
            }
            int temp = data.poll();
            swap();
            return temp;
        }

        private void swap() {
            Queue<Integer> temp = data;
            data = help;
            help = temp;
        }
    }

    public static void main(String[] args) {
        twoQueneStack t = new twoQueneStack<Integer>();
        t.push(10);
        t.push(1);
        t.push(-10);
        t.push(5);
        t.push(2);
        System.out.println(t.peek());
        t.poll();
        t.poll();
        System.out.println(t.peek());
    }
}

10. 双栈实现队列

一个push栈和一个pop栈，把push的所有东西压入pop，然后peek，pop即可。

import java.util.Stack;

public class 双栈实现队列 {
    public static class twoStack{
        private Stack<Integer> stackpush;
        private Stack<Integer> stackpop;

        public twoStack() {
            stackpop = new Stack<>();
            stackpush = new Stack<>();
        }

        public void push(int obj){
            stackpush.push(obj);
        }

        public int pop(){
            if(stackpop.isEmpty() && stackpush.isEmpty()){
                throw new RuntimeException("队列为空");
            }else if(stackpop.isEmpty()){
                while (!stackpush.isEmpty()){
                    stackpop.push(stackpush.pop());
                }
            }
            return stackpop.pop();
        }
        public int peek(){
            if(stackpop.isEmpty() && stackpush.isEmpty()){
                throw new RuntimeException("队列为空");
            }else if(stackpop.isEmpty()){
                while (!stackpush.isEmpty()){
                    stackpop.push(stackpush.pop());
                }
            }
            return stackpop.peek();
        }
    }

    public static void main(String[] args) {
        twoStack twoStack = new twoStack();
        twoStack.push(1);
        twoStack.push(21);
        twoStack.push(13);
        twoStack.push(14);
        twoStack.push(15);
        System.out.println(twoStack.pop());
        System.out.println(twoStack.peek());
        System.out.println(twoStack.peek());
    }
}
